Что делает макрос? При указании любого числового значения в пределах системы координат, опускает стол на указанное значение сразу после завершения снятия карты стола через веб интерфейс. Указание числа не в пределах системы координат выдаст ошибку, ничего не врежется.

Пример:
SORKIN_METHOD_BED_CALIBRATE DISTANCE=0 - отключение, работает стандартная калибровка.
SORKIN_METHOD_BED_CALIBRATE DISTANCE=100 - опускает стол на 100мм.

После перезапуска принтера по умолчанию работает стандартная калибровка.
